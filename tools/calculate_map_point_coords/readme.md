## Calculate Point Feature Coordinates The Easy Way!

This tool allows the user to calculate UTM and BC Albers X,Y coordinates for an input layer or feature class fast and easy! 

***Please see the script doc-string for further details.***

---
### How to Modify

It should be easy to modify to suite one's purposes, especially since the tool only has 1 parameter.

You will want to look at:
- The expected fields (that should exist in the input point feature class)
- The coordinate systems

Basically, look at lines 51 to 63 to start and then go from there.
